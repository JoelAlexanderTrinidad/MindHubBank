### Información General
***
App web homebanking para gestionar cuentas bancarias de manera online. Creado con:
  * Backend
    - Java
    - Spring boot
    - JPA | Hibernate
    - Spring security
  * Frontend
    - HTML | Css | js
    - Vue js
